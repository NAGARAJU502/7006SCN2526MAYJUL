## Project Overview

This project analyses the NYC Yellow Taxi dataset using PySpark and machine learning techniques. The study focuses on data preprocessing, feature engineering, predictive modelling, scalability analysis, model explainability, and business intelligence visualisation.

## Dataset

NYC Yellow Taxi Trip Records (January–April 2024)
