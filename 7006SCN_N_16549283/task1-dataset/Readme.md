# Task 1 – Problem Definition and Dataset Description

## Objective

The objective of Task 1 is to introduce the project problem, justify the dataset selection, and demonstrate that the dataset satisfies Big Data requirements.

## Dataset

- Dataset: NYC Yellow Taxi Trip Records
- Source: NYC Taxi and Limousine Commission (TLC)
- Period: January 2024 – April 2024
- Records: 13,069,067
- Columns: 19

## Activities Performed

- Loaded and explored the dataset using PySpark
- Examined schema and data types
- Analysed dataset dimensions
- Identified Big Data characteristics (5Vs)
- Defined the prediction problem

## Output

- Dataset overview
- Data schema
- Dataset statistics
- Big Data justification
