# Task 3 – Machine Learning Model Portfolio

## Objective

Develop and compare multiple machine learning algorithms for predicting high-tip taxi trips.

## Models Implemented

1. Logistic Regression
2. Decision Tree
3. Random Forest
4. Gradient Boosted Trees (GBT)

## Activities Performed

- Dataset splitting
- Model training
- Hyperparameter selection
- Prediction generation
- Model comparison

## Output

- Trained machine learning models
- Performance metrics
- Training time comparison
