# Task 2 – Data Engineering

## Objective

The objective of Task 2 is to prepare the raw taxi dataset for machine learning using PySpark data engineering techniques.

## Activities Performed

- Missing value analysis
- Data cleaning
- Feature engineering
- Creation of target variable (high_tip)
- Feature selection
- Feature scaling
- Pipeline construction using PySpark ML

## Pipeline Stages

1. StringIndexer
2. VectorAssembler
3. StandardScaler

## Output

- Cleaned dataset
- Engineered features
- ML-ready feature vectors
- Saved PySpark pipeline
